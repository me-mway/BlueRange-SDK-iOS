Overview

This class implements an analytical distance estimator based on the path-loss formula as described in: http://electronics.stackexchange .com/questions/83354/calculate-distance-from-rssi or Y. Wang, X. Yang, Y. Zhao, Y. Liu, L. Cuthbert, BRBluetooth positioning using rssi and triangulation methods, in: Consumer Communications and Networking Conference (CCNC), 2013 IEEE, 2013, pp. 837{842.
