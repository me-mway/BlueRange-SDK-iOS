BRBeaconMessageReporter.m

Overview

A beacon message reporter periodically reads the log of a {@link BRBeaconMessageLogger} instance and sends reports to a specified sender. The reports will be sent to the receiver every {@link #timeBetweenReportsInMs} milliseconds.
