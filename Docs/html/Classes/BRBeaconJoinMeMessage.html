BRBeaconJoinMeMessage.m

Overview

Beacon join me messages are send by BlueRange SmartBeacons, whenever a beacon is able to connect with another one. These messages contain useful information about a beacon which can be used to identify a beacon. Moreover, since they are sent regularly by a beacon these messages might be used for position estimation use cases like indoor navigation.
