Overview

This is the base class of all message processing elements. Each instance of this class can be interpreted as a node in a message stream processing graph, which can receive messages from a list of incoming edges and a can send the messages to all its receivers. By using this class as a base class of all message processing elements, it is possible to combine all elements to a flexible message processing architecture.
