Overview

A beacon message aggregator is a node in a message processing graph that merges equivalent beacon messages. Whenever the aggregator receives a message from a sender, the message will be added to a matching beacon message aggregate or, if a such an aggregate does not exist, it will be added to a new aggregate instance. Each aggregate has a predefined duration. After this duration all messages contained inside an aggregate will be combined to a new beacon message that is identical to the first message in the aggregate. However to make further message processing more stable, message properties like the RSSI value will be averaged by using a moving average filter. The resulting beacon message will be sent to all receivers of the node.
