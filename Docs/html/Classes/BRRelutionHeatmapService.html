Overview

This class periodically sends status reports to Relution. The reports contain the received signal strength of all beacons for all time intervals, since this service has been started. Each time interval has a maximum duration of 3 seconds. A report will be sent, whenever the 50th Join me message was received or no beacon has been detected in the last 5 seconds.
