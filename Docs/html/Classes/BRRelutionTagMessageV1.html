Overview

A Relution Tag message is a Relution specific advertising message that can be delivered by BlueRange SmartBeacons. A Relution tag message contains one or more tag identifiers. The concept of Relution Tag messages is designed for apps that do not require internet access but want to react to tags, that can be assigned dynamically to a beacon using the Relution platform.
