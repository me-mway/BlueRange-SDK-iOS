Overview

A beacon message queued stream node is a node in a message processing graph that queues all incoming beacon messages. The queue has a maximum size of {@link #maximumSize}. The messages will not be delivered to any receiver. However, they can be pulled by calling the {@link

pullBeaconMessage} method.
