Overview

A beacon message passing stream node is a node in a message processing graph that passes the incoming messages from all senders to all receivers. Before they are passed, however, an instance of a subclass can override the {@link #preprocessMessage} method to modify or analyze the data before passing it to its receivers. After the message has been passed to its receivers, the {@link #postprocessMessage} method will be called (which can also be overridden by a subclass) to do some postprocessing.
