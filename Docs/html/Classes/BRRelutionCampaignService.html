Overview

A Relution campaign service can trigger actions defined in a Relution campaign. Internally this class uses the {@link BRBeaconMessageActionTrigger} of the SDK’s core layer. The action registry obtains all action informations from the Relution system. To start the trigger, just call the {@link #start} method. The start method will periodically update a list of iBeacons each 10 seconds. In future versions actions will also be triggered when Relution tags will be received..
