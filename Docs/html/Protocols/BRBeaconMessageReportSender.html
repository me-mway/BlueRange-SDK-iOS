BRBeaconMessageReportSender Protocol Reference

Declared in BRBeaconMessageReportSender.h

Overview

A beacon message report sender is used by a {@link BRBeaconMessageReporter} instance to periodically send reports to a receiver. Implementations of this interface might deliver the report to a webservice or to another node in the message processing graph.
